## 一、先搞清楚你要住在哪里

KL很大，不同区域差异悬殊。选区域的核心逻辑：**离你工作/上学的地方近，或者在地铁沿线。**

### 华人聚集 / 中文通用区域

**Chow Kit（周溪）**
- 特点：华人最密集，中餐馆多，生活气息浓
- 价格：合租 RM 400–700/月，整租 RM 1,200–1,800/月
- 缺点：部分区域治安一般，夜晚注意安全

**Petaling Street（茨厂街）周边**
- 特点：唐人街，中文环境极好，采购方便
- 价格：与Chow Kit相近
- 缺点：交通拥挤，停车难

**Kepong**
- 特点：华人老社区，生活配套成熟
- 价格：稍低，整租约 RM 1,000–1,600/月
- 优点：超市、菜市场、华人餐馆齐全

---

### 白领 / 外籍人士聚集区

**Mont Kiara**
- 特点：外籍人士聚集，国际学校多，英文/中文均通
- 价格：整租 RM 2,500–5,000/月
- 适合：有家庭、小孩需要上国际学校的人群

**Bangsar**
- 特点：高档住宅区，餐厅酒吧多，生活品质高
- 价格：整租 RM 2,000–4,000/月
- 适合：收入较高的单身白领

**KLCC周边（双子塔附近）**
- 特点：市中心，交通便利，步行可达各大商场
- 价格：整租 RM 2,000–6,000/月（服务公寓为主）
- 适合：商务出行频繁的人

---

### 性价比之选

**Cheras**
- 特点：华人多，生活成本低，地铁可达市区
- 价格：合租 RM 350–600/月，整租 RM 900–1,500/月

**Setapak / Wangsa Maju**
- 特点：年轻人聚集，价格实惠
- 价格：整租约 RM 800–1,200/月

**Puchong**
- 特点：卧城，离市区稍远，但价格很低
- 价格：整租约 RM 700–1,200/月
- 缺点：公共交通稍弱，最好有车

## 四、租房前必须问清楚的7个问题

1. **租金包含什么？** 水电自付还是包含？有没有管理费（maintenance fee）？
2. **押金多少？** 正常是1–2个月租金，超过就问清楚
3. **合同期多长？** 一般1年起，短租通常更贵
4. **能否提前退租？** 违约金怎么算？
5. **有没有家具/家电？** 空房还是带家具？
6. **房子现状如何？** 入住前要拍照存证，避免退租时纠纷
7. **房东是否在马来西亚？** 要求视频通话确认真实身份

## 五、签合同注意事项

马来西亚租房合同一般是英文，篇幅较长，但以下几点务必看清楚：

| 条款 | 注意点 |
|------|-------|
| 租期 | 起止日期是否与约定一致 |
| 租金 | 金额、付款方式、涨租条款 |
| 押金 | 金额、退还条件、扣押金情形 |
| 维修责任 | 大修归房东，小修（通常低于RM 150）归租客 |
| 转租 | 是否允许转租/带朋友合住 |
| 违约 | 提前退租的罚款金额 |

**强烈建议：** 签合同前，请南洋通的中文翻译帮你过一遍，避免日后纠纷。

## 六、入住清单

- [ ] 拍下每个房间、每件家具的现状（含损坏处）
- [ ] 记录水表、电表初始读数
- [ ] 测试所有灯、水龙头、空调、热水器
- [ ] 确认紧急出口和门锁情况
- [ ] 把照片发给房东确认

**这些照片是你退租时保全押金的关键证据。**
